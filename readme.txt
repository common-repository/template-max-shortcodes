== Description ==

Template Max Shortcodes is a shortcode plugin that displays posted articles and menus.

You can display the posts by adding `[tmax_posts]` shortcode to the post or page.

== Installation ==

1. Upload `template-max-shortcodes.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[tmax_posts]` in your post or pages
